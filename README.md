# MCP RAG Memory Project

A Retrieval-Augmented Generation (RAG) system with memory capabilities using MCP (Model Context Protocol).

## Structure

- `src/` - Core source code
- `docs/` - Sample documents for RAG ingestion
- `tests/` - Unit tests
- `data/` - Persistent storage (ChromaDB, memory DB)

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env  # fill in your API keys
```
